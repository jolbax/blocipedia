# A Node.JS based Wikipedia clone
This project uses following libraries:
- Server: Express (flash, session, validation)
- Frontend: EJS, Markdown-it
- Authentication: Passport (Local and JWT)
- Mailing: Sendgrid
- Payment: Stripe
- ORM: Sequelize
- Testing: Jasmine, Faker, Request
